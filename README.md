# 🎯 Interactive Spotlight Mouse Animation

A modern interactive spotlight animation built with **HTML, CSS, JavaScript, and GSAP**.

This project demonstrates a premium mouse-follow interaction where an infinitely scrolling image strip follows the cursor while the text reacts with smooth lifting and dynamic wake effects. It is a great reference for learning advanced frontend animation techniques and interaction design.

---

## ✨ Features

* 🎯 Smooth mouse-follow spotlight effect
* ♾️ Infinite horizontal image marquee
* 🖱️ Cursor-based vertical movement
* 📝 SplitText line-by-line animation
* 🌊 Dynamic wake/ripple effect on text
* ⚡ Smooth interpolation (LERP) based animation
* 📱 Responsive layout
* 🚀 Built using vanilla JavaScript and GSAP

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP
* GSAP SplitText Plugin
